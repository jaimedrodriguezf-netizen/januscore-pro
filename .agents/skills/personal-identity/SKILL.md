## Application Decision System

Use this decision system after analyzing a job posting:

### Aplicar inmediatamente
- Match estimado >= 75%.
- Los gaps no son críticos o se pueden defender como transferibles.
- Existe evidencia real en Proof Bank o experiencia laboral para los requisitos centrales.

### Aplicar + estudiar rápido
- Match estimado entre 55% y 74%.
- Hay gaps técnicos importantes, pero se pueden cubrir con ruta de 7-14 días.
- El usuario tiene experiencia transferible o proyectos que justifican la postulación.

### No priorizar
- Match estimado < 55%.
- Requiere experiencia obligatoria no demostrable, certificación excluyente o seniority muy alejado.
- El tiempo de cierre de gaps es mayor al beneficio probable de aplicar.

Always include: decisión, perfil recomendado, CV a usar, evidencias, gaps, recursos verificados y material listo para copiar.
